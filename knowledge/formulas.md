# Important Math Formulas & Identities (JEE Level)

## Algebra

- Quadratic formula: For ax² + bx + c = 0 → x = [-b ± √(b² - 4ac)] / (2a)
- Arithmetic Progression: nth term → aₙ = a + (n-1)d
- Geometric Progression: nth term → aₙ = a·r^(n-1)
- Sum of first n naturals: 1 + 2 + ... + n = n(n+1)/2
- Sum of squares: 1² + 2² + ... + n² = n(n+1)(2n+1)/6

## Probability

- P(A∪B) = P(A) + P(B) - P(A∩B)
- P(A|B) = P(A∩B) / P(B)   (conditional probability)
- Bayes' theorem: P(A|B) = [P(B|A)·P(A)] / P(B)
- Independent events: P(A∩B) = P(A)·P(B)
- Binomial probability: P(X=k) = C(n,k) · pᵏ · (1-p)^(n-k)

## Calculus (Basic)

- Derivative of xⁿ → n·x^(n-1)
- Derivative of sin x → cos x
- Derivative of cos x → -sin x
- Derivative of eˣ → eˣ
- Derivative of ln x → 1/x
- Product rule: (uv)' = u'v + uv'
- Quotient rule: (u/v)' = (u'v - uv') / v²
- Chain rule: (f(g(x)))' = f'(g(x)) · g'(x)

## Linear Algebra (Basics)

- Determinant of 2×2 matrix [[a,b],[c,d]] → ad - bc
- Matrix multiplication (compatibility rule)
- Properties of transpose: (Aᵀ)ᵀ = A, (AB)ᵀ = BᵀAᵀ
